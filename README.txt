INSTRUCCIONES PARA VER EL LOGO EN EL PDF

Para que la imagen del logo se muestre correctamente en el PDF,
es necesario abrir el proyecto usando un servidor local.

Pasos:
1. Instalar Python (si no lo tienes).
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar:
   python -m http.server 8000
4. Abrir en el navegador:
   http://localhost:8000/index.html
5. Agregar tus clases y descargar el PDF.
